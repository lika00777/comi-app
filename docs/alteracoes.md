## Convenções de Versionamento

- **v0.x.x** - Desenvolvimento inicial
- **v1.x.x** - Primeira versão de produção
- **Incremento Major (X)** - Alterações incompatíveis com versões anteriores
- **Incremento Minor (Y)** - Novas funcionalidades compatíveis
- **Incremento Patch (Z)** - Correções de bugs
